## Features

- Add tasks using the input box and ADD button
- Check off tasks with a checkbox (strikes through when done)
- Delete individual tasks with the DELETE button
- Remove a task by name using the REMOVE button
- Responsive layout — works on desktop, tablet, and mobile

## Tech Stack

- HTML
- CSS (Flexbox + CSS Grid)
- JavaScript (Vanilla)

## How to Run

1. Download or clone the project folder
2. Make sure `AVATAR.png` is in the same folder
3. Open `index.html` in any browser
4. Start adding tasks!

## How It Works

- Type a task in the input box and click **ADD** — it appears in the list with a checkbox and DELETE button
- Click the **checkbox** to mark a task as done (text gets struck through)
- Click **DELETE** on any row to remove that specific task
- Type a task name and click **REMOVE** to remove it by name
